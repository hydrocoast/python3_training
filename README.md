# Requirements
- Python 3.6.x
- numpy
- scipy
- matplotlib
- OpenCV2
- wget (for Python)
- NetCDF4

# Usage
- Install the required libraries
- Clone this repository
```bash
git clone https://github.com/hydrocoast/python3_training
```
- Excecute the python scripts
```bash
python EX1_peaks_surf.py
python EX2_rand_hist.py
...
```
